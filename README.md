# Five Nights at Freddy’s 1 – Exact Browser Replica

This project is a **faithful 1:1 recreation** of **Five Nights at Freddy’s 1**, fully playable on both **desktop and mobile browsers**. Developed using **HTML5** and **JavaScript**, it faithfully replicates the original gameplay mechanics, visuals, and user experience.

## Features
- Exact recreation of the original FNaF 1 gameplay  
- Fully browser-based, requiring no installation  
- Optimized for both PC and mobile devices  
- Preserves original mechanics, jumpscares, and night progression  

## How to Play
1. Open the game in a modern web browser: [Play Here](https://testingfivenightsatfreddys.vercel.app)  
2. Monitor the security cameras to track animatronic movement.  
3. Use doors and lights strategically to prevent animatronics from entering the office.  
4. Manage power carefully—running out of power ends the night.  
5. Survive all five nights to complete the game.  

## Controls
- **PC:** Use mouse or keyboard for camera navigation, doors, and lights.  
- **Mobile:** Tap on-screen buttons to control cameras, doors, and lights.
